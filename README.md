# 💊 MediVerify — AI-Powered Medicine Verification

> Final Year Project | Software Engineering | 2024-2025

MediVerify is a web application that uses **Claude AI** to instantly verify whether a medicine is **Legitimate ✅**, **Fake ❌**, or **Expired ⚠️** — by analyzing a photo of the packaging or a registration/batch code.

## ✨ Features

- **📷 Image Upload** — Drag & drop or browse a photo of the medicine back label
- **🔢 Code Entry** — Enter batch number or registration code manually
- **🤖 AI Analysis** — Claude AI reads and verifies medicine details
- **✅ Three Verdicts** — Legit, Fake, or Expired with confidence score
- **🔍 5 Verification Checks** — Registration, batch, expiry, manufacturer, packaging
- **🗂️ Scan History** — All scans saved locally with filters and stats
- **📱 Mobile Responsive** — Works perfectly on phone and desktop
- **🍞 Toast Notifications** — Friendly feedback on every action
- **⚠️ Smart Error Handling** — Helpful error messages with retry option

## ⚙️ How to Run Locally

### Prerequisites
- Node.js v22 or higher
- Claude API key from [console.anthropic.com](https://console.anthropic.com)

### Steps

**1. Clone or download the project**
```bash
cd "Medi Verify/mediverify"
```

**2. Install dependencies**
```bash
npm install
```

**3. Create .env file**

Create a file named `.env` in the root of the project:
```
VITE_CLAUDE_API_KEY=sk-ant-your-api-key-here
```

**4. Start development server**
```bash
npm run dev
```

**5. Open in browser**
```
http://localhost:5173
```

## 🧠 How the AI Works

MediVerify sends medicine information to Claude AI which:

1. **Reads the image** using Claude Vision (OCR)
2. **Extracts details** — batch number, expiry date, registration code, manufacturer
3. **Analyzes patterns** — checks if codes match known formats
4. **Returns a verdict** — Legit, Fake, or Expired with confidence score and detailed checks

> **Note:** The AI analyzes patterns and formats. For a production system, direct integration with DRAP's official database would provide fully verified results.

---

## 📊 Verdict Explanations

| Verdict | Meaning |
|---------|---------|
| ✅ **LEGIT** | Medicine appears genuine, registered, and within expiry date |
| ❌ **FAKE** | Medicine shows signs of being counterfeit or unregistered |
| ⚠️ **EXPIRED** | Medicine is genuine but past its expiry date |

## 🚀 Deployment (Netlify)

4. Build settings:
